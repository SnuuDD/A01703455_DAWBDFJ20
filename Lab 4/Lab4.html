Theatre Square in the capital city of Berland has a rectangular shape with the size n × m meters. On the occasion of the city's anniversary, a decision was taken to pave the Square with square granite flagstones. Each flagstone is of the size a × a. What is the least number of flagstones needed to pave the Square? It's allowed to cover the surface larger than the Theatre Square, but the Square has to be covered. It's not allowed to break the flagstones. The sides of flagstones should be parallel to the sides of the Square.
Input
The input contains three positive integer numbers in the first line: n,  m and a (1 ≤  n, m, a ≤ 109).
Output
Write the needed number of flagstones.

¿Qué métodos tiene el objeto Date?

Date.UTC() toma los parámetros de la fecha delimitados coma y devuelve el número de milisegundos entre las 00:00:00 del 1 de enero de 1970 (hora universal) y la hora que especifique.
Date.parse() Transforma una cadena con la representación de una fecha y hora, y devuelve el número de milisegundos desde las 00:00:00 del 1 de enero de 1970, hora local.
Date.prototype.getDate() El valor devuelto por getDate() es un número entero, entre 1 y 31, que representa el día del mes para la fecha dada según la hora local.
Date.prototype.getTime() El método getTime() devuelve el valor numérico correspondiente a la hora para la fecha especificada según la hora universal
Date.prototype.getTime() El método getTime() devuelve el valor numérico correspondiente a la hora para la fecha especificada según la hora universal
Date.prototype.toISOString() El método toISOString() devuelve una cadena en el formato simplificado extendido ISO (ISO 8601), que siempre mide 24 caracteres de largo: YYYY-MM-DDTHH:mm:ss.sssZ. El uso horario no tiene retraso respecto a UTC, como lo denota el sufijo "Z".

¿Qué métodos tienen los arreglos?

Array.isArray() determina si el valor pasado es un Array.
Array.of() crea una nueva instancia Array con un número variable de elementos pasados como argumento, independientemente del número o del tipo.
Array.push() El método push() añade uno o más elementos al final de un array y devuelve la nueva longitud del array.
Array.prototype.shift() El método shift() elimina el primer elemento del array y lo retorna. Este método modifica la longitud del array.
Array.prototype.lastIndexOf() El método lastIndexOf() devuelve el último índice en el que un cierto elemento puede encontrarse en el arreglo, ó -1 si el elemento no se encontrara. El arreglo es recorrido en sentido contrario, empezando por el índice fromIndex.
